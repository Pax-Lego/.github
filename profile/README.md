## 📌 Descripción

Pax-Lego es un proyecto académico desarrollado en la asignatura Ingeniería de Software I, enfocado en aplicar metodologías ágiles y buenas prácticas de desarrollo mediante la construcción colaborativa de una solución de software.

## 🧠 Metodología de Trabajo

### ⚙️ Scrum
- Desarrollo iterativo por **Sprints**
- Sprint Planning
- Daily Scrum
- Sprint Review
- Sprint Retrospective

### 📋 Kanban
- Visualización del flujo de trabajo
- Control del progreso mediante tablero
- Limitación del trabajo en progreso (WIP)

---

## 🎯 Objetivos del Proyecto

- ✅ Aplicar principios de Ingeniería de Software
- ✅ Trabajar colaborativamente usando Git y GitHub
- ✅ Diseñar una solución software funcional
- ✅ Documentar el proceso de desarrollo
- ✅ Implementar metodologías ágiles reales
